## 项目简介

本仓库用于整理和展示我在人工智能通识课中的实验成果。  
作为一名非计算机专业的学生，我希望通过这些实验，逐步理解人工智能的基本思想、数据处理方法以及经典机器学习算法的实际应用。

目前仓库主要包含两个实验方向：

- **实验一：基于大模型的温度数据处理分析与城市时空差异探究**
- **实验二：经典机器学习算法实践**

---

## 实验内容

### 实验一：温度数据处理与城市时空差异分析

本实验围绕北京、南京、广州三座城市的逐小时温度数据展开，主要完成了以下工作：

- 对温度数据进行清洗与整理
- 基于合理规则判断异常值
- 采用合适方法填充缺失值
- 计算平均值、中位数、最大值、最小值和标准差等统计量
- 绘制温度变化折线图与温度分布小提琴图
- 分析不同城市在冬季气温变化上的差异与规律

### 实验二：经典机器学习算法实践

本实验主要围绕监督学习中的回归与分类任务展开，使用 Python 和经典机器学习库完成建模与评估。实验内容包括：

- 加载经典数据集
- 对数据进行标准化处理
- 划分训练集与测试集
- 完成回归任务与分类任务
- 对比不同模型的效果
- 使用图表展示模型预测结果与混淆矩阵

实验中涉及的数据集包括：

- 糖尿病数据集（`sklearn.datasets.load_diabetes`）
- 乳腺癌数据集（`sklearn.datasets.load_breast_cancer`）

实验中使用的模型包括：

- 线性回归
- 随机森林回归
- XGBoost 回归
- 逻辑回归
