# My-AI-Experiment

> 人工智能通识课实验仓库  
> 记录我在课程学习中完成的三个实验项目，包括数据分析、机器学习建模与深度学习实践。

---

## 项目简介

本仓库用于整理和展示我在人工智能通识课中的实验成果。  
作为一名非计算机专业的学生，我希望通过这些实验，逐步理解人工智能的基本思想、数据处理方法以及经典机器学习算法的实际应用。

目前仓库主要包含四个实验方向：

- **实验一：基于大模型的温度数据处理分析与城市时空差异探究**
- **实验二：经典机器学习算法实践**
- **实验三：深度学习——手写数字识别（全连接网络 + CNN）**
- **实验四：Ollama 与 OpenCode 本地 AI 编程助手部署**

---

## 实验内容

### 实验一：温度数据处理与城市时空差异分析

本实验围绕北京、南京、广州三座城市的逐小时温度数据展开，主要完成了以下工作：

- 对温度数据进行清洗与整理
- 基于合理规则判断异常值
- 采用合适方法填充缺失值
- 计算平均值、中位数、最大值、最小值和标准差等统计量
- 绘制温度变化折线图与温度分布小提琴图
- 分析不同城市在冬季气温变化上的差异与规律

相关文件：

- `实验一/实验1.md`
- `实验一/基于大模型的温度数据处理分析与城市时空差异探究实验报告.md`
- `实验一/基于大模型的温度数据处理分析与城市时空差异探究实验报告.pdf`

### 实验二：经典机器学习算法实践

本实验主要围绕监督学习中的回归与分类任务展开，使用 Python 和经典机器学习库完成建模与评估。实验内容包括：

- 加载经典数据集
- 对数据进行标准化处理
- 划分训练集与测试集
- 完成回归任务与分类任务
- 对比不同模型的效果
- 使用图表展示模型预测结果与混淆矩阵

实验中涉及的数据集包括：

- 糖尿病数据集（`sklearn.datasets.load_diabetes`）
- 乳腺癌数据集（`sklearn.datasets.load_breast_cancer`）

实验中使用的模型包括：

- 线性回归
- 随机森林回归
- XGBoost 回归
- 逻辑回归

相关文件：

- `实验二/test.py`

### 实验三：深度学习——手写数字识别（全连接网络 + CNN）

本实验基于 PyTorch 框架完成 MNIST 手写数字识别任务，从零构建深度学习模型并完成训练与评估。实验涵盖全连接神经网络（FCNN）和卷积神经网络（CNN）两种模型，并对二者进行比较分析。实验内容包括：

- 使用 PyTorch 搭建全连接神经网络和卷积神经网络
- 加载并预处理 MNIST 手写数字数据集
- 设计网络结构、设置损失函数和优化器
- 训练网络并绘制损失曲线、准确率曲线
- 评估模型性能，生成混淆矩阵和分类报告
- 可视化预测结果与错误样本
- 对比分析 FCNN 与 CNN 的差异

相关文件：

- `实验三/test.py`
- `实验三/深度学习——手写数字识别实验报告.md`
- `实验三/实验三-实验手册（深度学习版）_手写数字识别_全连接网络-上传).pdf`

**实验结果**：FCNN 测试准确率 97.26%，CNN 测试准确率 98.78%

### 实验四：Ollama 与 OpenCode 本地 AI 编程助手部署

本实验完成本地大语言模型运行环境的部署与配置，以及 AI 编程助手的使用实践。实验内容包括：

- 在本地计算机上安装并配置 Ollama 大语言模型运行环境
- 安装 OpenCode 并将其连接到本地 Ollama 模型，形成可在终端中使用的 AI 编程助手
- 阅读 OpenCode 中文说明文档，理解其安装方式、Agent 模式切换和基础使用方法
- 使用 OpenCode 辅助撰写本次实验报告，亲身体验本地 AI 编程助手的工作方式
